# About

I am an independent researcher based in Seoul, Republic of Korea, working at the
intersection of financial services, AI/ML systems, and regulatory compliance.

## Background

- **GARP** Financial Risk Manager (FRM)
- Career in Korean financial sector spanning credit and market risk analysis,
  regulatory compliance, MyData licensing, big data platform operations, data
  science projects, and recommendation system management

## Current work

The [Heterogeneous Expert PLE project](https://github.com/bluethestyle/aws_ple_for_financial)
is a three-person, personal-time research effort started in early 2026.
Papers are on Zenodo under CC BY 4.0; code under MIT License.

The system uses Claude Code (Anthropic) as the primary development partner
— roughly 90% of the codebase and both Zenodo preprints were built through
human-AI collaboration. The [Advanced Claude Code Usage Patterns](https://github.com/bluethestyle/aws_ple_for_financial#advanced-claude-code-usage-patterns)
section of the main repository documents the workflow patterns used.
